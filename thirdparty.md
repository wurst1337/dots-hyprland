# Third party list

In this file specified a list of third-party software, which didn't (and is) written in a [README](README). 

## Primary
- **Hyprland**, a primary tiling manager.
- **Kitty**, a terminal witch Hyprland and I uses.
- **Rofi**, a launcher.
- **Waybar**, a bar.
- **SwayNC**, a notification center.
- **Slurp, Swappy and Grim**, a screenshots tools.
- **awww**, a wallpaper daemon.
- **pywal**, a themes generator.
- **Hypridle, Hyprlock**, for a lock.

## Other
